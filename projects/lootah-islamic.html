Lootah Islamic

Methods & journey

The Lootah Group wanted to bring their Islamic products brand digital — a clean, trusted mobile experience where UAE customers could browse and purchase Sharia-compliant goods with confidence. My role was to design and build the entire backend that powers the customer-facing iOS and Android apps.

The challenge

The app needed to handle a rich product catalog with Islamic certification metadata, support both Arabic and English content, and integrate seamlessly with existing Lootah Group inventory and fulfilment systems. All of this had to be delivered with a clean, documented API that the Flutter team could build on quickly.

  • Bilingual API responses (Arabic / English) based on request locale
  • Product catalog with Islamic certification labels and category filters
  • Secure user authentication with OTP verification (UAE phone numbers)
  • Order management: create, track, cancel, return flows
  • Payment gateway integration (online and cash on delivery)
  • Admin panel for product management, orders, inventory, and customer support

The architecture

Laravel as the backend framework with a RESTful API contract documented in Postman. MySQL for the relational data layer with a product-category tree, order status machine, and audit logging on all critical operations. Redis for session management and caching category/product lists to keep API responses under 100ms.

The admin panel was built with Laravel Livewire — giving the operations team a fast, reactive UI for order processing and inventory updates without the complexity of a separate SPA build.

What shipped

A production-ready Laravel API serving the iOS and Android apps, a full Livewire admin panel, OTP-based registration, complete order lifecycle management, and live product catalog with real-time inventory sync.

The outcome

App live in the App Store and Google Play. Customers can browse Islamic products, place orders, and track delivery — all in a trusted, Lootah-branded experience. The admin team moved from manual spreadsheet tracking to a real-time order dashboard with zero onboarding friction.
