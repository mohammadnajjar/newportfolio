Stockart

Methods & journey

Stockart is a curated UAE art marketplace — a place where emerging and established artists can list their work and reach genuine buyers directly through a mobile app. I was brought in to architect and build the entire backend, from the data model to the production API to the admin operations panel.

The challenge

Art marketplaces have complex data relationships: each artwork belongs to an artist, exists as a unique piece (or limited edition), has an ownership history, and may have multiple images. The purchase flow needs to handle real-money payments, commission splits between the platform and the artist, and secure delivery coordination — all through a clean mobile API.

  • Artist onboarding flow: profile, portfolio, verification, payout details
  • Artwork listings: multi-image, categories, dimensions, medium, edition type
  • Buyer experience: browse, save, buy, track order, request return
  • Payment integration with platform commission split on each transaction
  • Admin panel: artist verification, listing moderation, order management, payouts
  • Push notifications: order updates, new follower, artwork sold alerts

The architecture

Laravel with MySQL — artwork data modeled with a polymorphic media library for flexible multi-image support. The payment flow uses Stripe with a Connect account per artist, allowing automatic commission splits on charge without any manual reconciliation. Order status is managed through a state machine with event-driven notifications via Firebase Cloud Messaging.

The admin panel handles artist verification, content moderation, order dispute resolution, and payout monitoring — built with Filament for a fast, modern back-office experience.

What shipped

Production API on stockart.ae serving the iOS and Android apps. Artist onboarding, multi-image artwork listings, Stripe payment with commission splits, full order lifecycle, push notifications, and a Filament admin panel.

The outcome

Artists list their work and receive payments directly to their linked accounts with zero manual reconciliation. Buyers browse a curated catalog and checkout with full delivery tracking. The platform team manages everything through the admin panel without touching code.
